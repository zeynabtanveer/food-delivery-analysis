Food Delivery Data Analysis

 Project Overview
Analysis of 1000 food delivery orders using 
Python, PostgreSQL and Power BI.

 Tools Used
- Python (Pandas, Matplotlib, Seaborn)
- PostgreSQL + pgAdmin
- Power BI
- Jupyter Notebook

 Key Insights
- Chinese cuisine most popular
- Card payment = highest loyalty (54.5%)
- Evening peak order time
- Average delivery time: 44.94 minutes
- Repeat customer rate: 50.5%
 Files
 food_delivery_LIGHT_DIRTY.xlsx (dataset)
- food_delivery.ipynb → Python analysis
- food_delivery.sql → SQL queries
- ` fooddelivery.pdf` → Power BI screenshot
